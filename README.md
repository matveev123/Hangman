#Виселица

Игра реализованная в консоли по разгадванию слова по 1 букве. 1 буква = 1 попытка. Есть всего 7 попыток. При неуспехе попытки угадать букву рисуется часть фигуры "висельника".

#Оссобености

- все буквы только русские, в загаданых словах символы написаны неКАПСОМ;
- учтены попытки отказа засчитвать ошибку при повторе буквы в рамках угадвания 1 слова
- учтены попытки отказа засчитвать букву ранее введенную в рамках угадывания 1 слова
- рисование реализовано "символами"
- включена опция перезапуска игры
- после перезапуска секретное слово сменяется
- все слова хранятся в отдельном файле - "словаре"

## Как играть

1. Выбор в стартовом меню при помощи ввода в консоль требуемых символов для продолжения игры
2. следуя текстовым подсказкам угадывать букву в слове
3. Выбор в меню при помощи ввода в консоль требуемых символов для завершения или продолжения игры
